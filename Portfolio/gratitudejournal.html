Gratitude Journal

Mermaid Labs: Happiness Apps for Mobile and Wearable Tech

Unlock Your Happiness & Spread Gratitude

Background

The original idea for this project came about when I discovered the gratitude apps on the Apple store were priced between .99-$4.99. I was frustrated to find that developers were monetizing mental health solutions so I wanted to publish a similar, but improved app for free. I wanted to keep the main elements of tracking gratitude the same, but I wanted to include a few new features that these apps didn't have. Specifically, I wanted to make the app compatible with the Apple Watch, by generating arbitrary positivity notifications throughout the day.

This project was created for Mermaid Labs LLC, a developer group creating mobile and wearable apps with an emphasis on free, open source products.

Process

I initially rapid prototyped the initial flow of the mobile app and did two informal usability tests. This led to large changes to the overall design of the application, as well as adding more informative user interface text and a completely different user login method. Then, I used Sketch and Invision to flesh out the visual flow of the app and started conducting usability tests.

Outcome

The project is on hold until I can ship it onto the Apple App Store. I'm currently learning object-oriented programming and slowly moving onto Swift and XCode to learn how to develop the app myself.
